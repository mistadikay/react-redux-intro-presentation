This presentation is intended as introductory for people not familiar with react/redux ecosystem and its benefits.

## Development

```sh
npm install
npm start
```

Open a browser and hit [http://localhost:3000](http://localhost:3000)

## Build

`npm run build`

## Deploy to surge

`npm run deploy`
